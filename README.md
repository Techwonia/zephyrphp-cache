# ZephyrPHP Cache

Multi-driver caching module for [ZephyrPHP](https://zephyrphp.com).

## Features

- File cache driver
- Redis cache driver
- APCu cache driver
- Array cache driver (in-memory)
- TTL support
- Cache tags

## Installation

```bash
composer require zephyrphp/cache
```

> **Note:** This package is built for ZephyrPHP and is not compatible with Laravel or other PHP frameworks.
